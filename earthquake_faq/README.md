## 簡介

本知識庫將台灣中央氣象署出版的「地震百問」PDF 轉化為分章節的 Markdown 格式，便於查詢與引用。共收錄100個地震相關問答，涵蓋從地球科學基礎到地震防災應變的完整知識。
