## 六、報告品質評估 / Report Quality Assessment

### 6.1 品質指標 Q 值分布

Q 值由 Earthworm 內部定義，負值通常代表初步估計，Q=0 表示收斂良好：

| Q 值 | 報告數 | 說明 |
|------|--------|------|
| Q = 0 | 9 | 品質最佳（解收斂） |
| Q = -1 | 6 | 良好 |
| Q = -2 | 18 | 良好 |
| Q = -3 | 55 | 一般（最常見） |
| Q = -4 | 30 | 可接受 |
| Q = -5 | 15 | 偏低 |
| Q = -6 | 81 | 偏低（第二常見） |
| Q ≤ -7 | 5 | 低品質 |

> 📌 Q=-3 及 Q=-6 最為常見，兩者合計佔 **62%**。Q=-6 代表解的收斂程度有限，可能受限於觀測角度（Gap 角度大）或測站分布不均。

- **654 筆** `_f42` 報告對應約 160 個事件

- `_f42` 報告用於即時預警推播，最終 `.rep` 用於存檔與品質驗證
